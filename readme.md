# Key Manager CLI

A CLI for managing configuration in AWS Parameter Store.

## Commands:

Name: `download`
Alias: `read, get, d`
Description: Download the configuration from parameter store to run the application

Name: `upload`
Alias: `write, u, put`
Description: Upload the configuration to parameter store
